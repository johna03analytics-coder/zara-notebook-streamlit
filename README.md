# zara-notebook-streamlit
Curated and cleaned Zara dataset from kaggle and produced a streamlit dashboard using it.

##Analyzed:
- Identified best selling days
- Visualized sales over recorded time
- Visualized frequency of distribution of sales
- Created a table with total sales in each category
- Visualized table
- Curated a table of Average price per gender per category of purchase

##Tools used:
- pandas
- streamlit
- seaborn
- matplotlib

## findings:
- Identified top ten days of selling
- Analyzed relationship between genders and their average purchases
